Create your Scouting.io profile

Your online Scouting.io profile will become your passport to all web and mobile apps developed using our platform. This provides for a sign sign-on, profile, and billing for easier usage and better privacy controls.

Start Now

Single Sign-on

Using Oauth 2, users can sign-on and authorize an application to use their Scouting.io account.

Unified Billing

Billing information is stored by the user on Scouting.io and used for payment to all applications.

Shared Datasets

Common datasets are accessible through the API such as councils, areas, regions, position codes, merit badges, ranks, awards, training courses, and more.
